------------------------ИНСТРУКЦИИ-----------------------------------------------
Распакуйте папку на компьютер с установленным Python3

	---------WINDOWS-----------:
Запустите файл 'startserver.py' на выполнение.
Должно открытся и не закрыватся окно консоли.

Если у вас не установлен Python3, то можно скачать его по адресу:
https://www.python.org/ftp/python/3.5.1/python-3.5.1.exe

	----------LINUX------------:
Вначале добавьте права на выполнение (Allow executing file в Ubuntu) файлам:
	"startserver.py"
	"cgi-bin/index.py"
Это можно сделать в свойствах файла, например в Ubuntu это галочка
	"Allow executing file" на вкладке "Permissions"
или из консоли командой:
	"chmod +x имяФайла" 

После этого запустите файл "startserver.py" мышкой или из консоли командой
	"python3 pathtofile/startserver.py" (pathtofile это путь к файлу)

Должно открытся и не закрыватся окно консоли.

Обычно в линуксе Python3 уже установлен, если нет, то Python3 можно
установить командой:
	'sudo apt-get install python3'

-----------------------РАБОТА С ПРОГРАММОЙ----------------------------------------------
После того как сервер запустился, то есть окно консоли запустилось и не закрывается 
(пока никто не зашел на сайт, окно может быть пустым)

Откройте браузер и перейдите по адресу:

			http://localhost:8000/
